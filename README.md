# LifeLink – Connecting Donors 🩸

LifeLink is a full-stack Blood Bank Management System designed to connect blood donors with people in need and efficiently manage blood stock and blood requests.

## 🚀 Features

- Donor Registration
- Blood Group Search
- Blood Stock Management
- Blood Request System
- Dashboard
- Donor and Request Management

## 🛠️ Tech Stack

### Frontend
- React
- Vite
- Axios
- CSS

### Backend
- Node.js
- Express.js

### Database
- MongoDB
